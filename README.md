# Momentum Landing

## Overview
Momentum Landing is a Vibe Coding project built using Claude AI. The goal was to design a clean, modern landing page with clear hierarchy, intuitive navigation, and engaging visuals.

## Tools & Process
- **Design Tool**: Claude AI (Free Plan)
- **Process**: Ideation → Wireframe → Prototype → Refined Landing Page
- Focused on **UX principles** like clear CTAs, visual hierarchy, and responsive layout.
